Mathematical Theory

The calculus concepts that make our roller coaster simulation possible through derivatives, curvature, and continuity analysis.

Track Function f(x): Piecewise Shape of the Coaster

Physical Meaning: The height of the roller coaster track at each horizontal position x.

Four Track Segments (updated coefficients):

For 0 ≤ x < 5:
$$f_1(x) = 20 - 0.32 x^2$$
For 5 ≤ x < 15:
$$f_2(x) = 0.0064 (x - 10)^4 + 8.0$$
For 15 ≤ x < 25:
$$f_3(x) = 20.0 - 0.32 (x - 20)^2$$
For 25 ≤ x ≤ 35:
$$f_4(x) = 0.32 (x - 30)^2 + 4.0$$

Combined Piecewise Track:

$$ f(x) = \begin{cases} 20 - 0.32 x^2, & 0 \le x < 5 \\ 0.0064 (x-10)^4 + 8.0, & 5 \le x < 15 \\ 20.0 - 0.32 (x-20)^2, & 15 \le x < 25 \\ 0.32 (x-30)^2 + 4.0, & 25 \le x \le 35 \end{cases} $$

First Derivative f'(x): Slope & Direction

The first derivative represents the instantaneous slope (steepness) of the track.

Calculated Derivatives:

For 0 ≤ x < 5:
$$f'_1(x) = -0.64 x$$
For 5 ≤ x < 15:
$$f'_2(x) = 0.0256 (x - 10)^3$$
For 15 ≤ x ≤ 25:
$$f'_3(x) = -0.64 (x - 20)$$
For 25 ≤ x ≤ 35:
$$f'_4(x) = 0.64 (x - 30)$$

Curvature (κ) and Radius (ρ)

To determine safety (G-forces), we need the curvature of the track. Engineers often use the Radius of Curvature.

Curvature Formula (κ):

$$ \kappa(x) = \frac{|f''(x)|}{(1 + [f'(x)]^2)^{3/2}} $$

Radius of Curvature (ρ):

The radius of the imaginary circle that "kisses" the track at a specific point.

$$ \rho(x) = \frac{1}{\kappa(x)} = \frac{(1 + [f'(x)]^2)^{3/2}}{|f''(x)|} $$

Engineering Limitations: The "Jerk"

Academic Note on C² Continuity:

While our track is C¹ Continuous (smooth slope), it is not C² Continuous. At the connection points (e.g., x=5), the second derivative (curvature) jumps instantaneously from a constant value (parabola) to a variable value (quartic). In physics, this sudden change in acceleration is called a "Jerk." In real-world engineering, transition curves (like Euler spirals) are inserted between segments to smooth out these G-force spikes.
